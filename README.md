# Cubika

- 개요
  수박게임(Suika Game)을 모티브로한 큐브 퍼즐 게임
  큐브를 회전하고 떨어뜨리고 밀쳐서 같은 종류의 큐브를 합쳐 높은 점수를 얻는다.
